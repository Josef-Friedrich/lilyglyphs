The Feta font

This document is derived from the official LilyPond Notation Reference, and the original version of this file can be found at http://www.lilypond.org/doc/v2.15/Documentation/notation/the-feta-font.html

It is used as a documentation on the progress of the glyph coverage in the lilyglyphs LaTeX package.

The following symbols are available in the Emmentaler font and can (in LilyPond) be accessed directly using text markup with the name of the glyph as shown in the tables below, such as g^\markup {\musicglyph #"scripts.segno" } or \markup {\musicglyph #"five"}. For more information, see Formatting text.

The goal of the lilyglyphs package is to make these glyphs available for use in the continuous text in LaTeX documents. The ultimate goal is a complete coverage, but the project starts with the glyphs the initiator needs most.

Whenever a command covering a glyph has been implemented, an entry should be placed here (TBD: Explain syntax)

When all glyphs of a category have been covered please change the heading and the corresponding menu entry, appending "completed"
